## 📦 Technologies Used

- **Next.js 13+ App Router**
- **Tailwind CSS**
- **Axios**
- **PostgresSQL**
- **JavaScript**
- **JWT & Cookies**

## 🔧 Setup Instructions

1. **Clone the repo**
```bash
git clone https://github.com/07asn/Next-CRUD.git
cd next-crud
npm i
npm run dev
```
